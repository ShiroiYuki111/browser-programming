# Lecture 05 – DOM, Events, and Web APIs

## Features Implemented
1. **New Content:** Added an interactive Skills grid and a Projects section utilizing semantic HTML and CSS.
2. **Persistent Theme Toggle:** Upgraded the dark mode toggle to use the Browser's `localStorage`. The page now remembers the user's theme preference (`portfolio_theme`) even after refreshing or closing the browser.
3. **Dynamic Date Generation:** Used the JavaScript `Date` object to automatically generate and display the current date in a `YYYY-MM-DD` format at the bottom of the page.

## How to test:
- Click the "Toggle Dark Mode" button, then refresh the page. The site should remember your choice and stay in the theme you selected.
- Scroll to the very bottom of the page to verify the "Last updated" date matches today's actual date.
