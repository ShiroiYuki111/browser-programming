# Lecture 04 – JS Basics

## Features Implemented
1. **Feature A (Theme Toggle):** Added a button in the header that toggles a `.dark-mode` CSS class on the `<body>` and updates an `isDarkMode` state variable.
2. **Feature C (Portfolio Contact Quick Action):** Added a button in the footer that triggers an `alert()` box displaying contact information.

## How to Test
1. **Test the Theme:** Click the "Toggle Dark Mode" button in the header. The background should turn dark, the button text should change, and the change will be logged in the console.
2. **Test the Contact Action:** Scroll to the footer and click the "Quick Contact Info" button. An alert box will pop up on the screen, and the click event will be logged in the console. 
*(Remember to open DevTools F12 to see the `console.log` outputs!)*
